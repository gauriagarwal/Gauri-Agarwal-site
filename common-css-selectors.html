Commonly used CSS Selectors

CSS Selectors are the heart of CSS. They allow you to specify what specific parts of a webpage you want to select. In this blog, we will take a look at some of the most commonly used and helpful CSS selectors. Element selector: This selector allows you to select all occurrences of an HTML element. For eg.

p { padding: 20px; } h1 { padding: 16px; }
Id selector: This selector allows you to select an element with a specific id. If multiple elements share the same id attribute then the styles will be applied to the first element that has that.

Hello

 To style the above paragraph, the selector will be "#" + id attribute value i.e. #intro #intro { color: blue; }
Class selector: This selector allows you to select all elements that have a class attribute value. For eg. To style all of the paragraphs in a webpage with the class attribute as "red", you will have to use "." + "red" .red { color: red; }
Universal selector: You can use this selector to apply styles to each and every tag on the webpage. One common use of this tag is to remove the default margins and paddings of each tag. The universal selector is "*" * { padding: 0; margin: 0; }
